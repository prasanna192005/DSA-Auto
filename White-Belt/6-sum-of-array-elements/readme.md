# Sum of Array Elements

**Problem Description:**

Given an integer array `arr` of size `n`, write a program to calculate the sum of all its elements.

**Constraints:**

* `1 <= n <= 1000`
* `-1000 <= arr[i] <= 1000` for all `i` from 0 to `n-1`

**Example:**


Input: arr = [1, 2, 3, 4, 5]
Output: 15
