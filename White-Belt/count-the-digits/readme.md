# Count the Digits

**Problem Description:**

Given a non-negative integer, count the total number of digits in that integer.

**Constraints:**

* The input integer will be between 0 and 10<sup>9</sup> (inclusive).

**Examples:**

* Input: 12345
  Output: 5
* Input: 0
  Output: 1
* Input: 9876543210
  Output: 10
