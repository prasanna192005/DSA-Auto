# Find the GCD

**Problem Description:**

Given two non-negative integers, find their greatest common divisor (GCD) using the Euclidean algorithm.

**Constraints:**

* 0 ≤ a ≤ 10<sup>6</sup>
* 0 ≤ b ≤ 10<sup>6</sup>

**Examples:**

* **Input:** a = 48, b = 18
  **Output:** 6
* **Input:** a = 10, b = 5
  **Output:** 5
* **Input:** a = 0, b = 10
  **Output:** 10
* **Input:** a = 10, b = 0
  **Output:** 10
